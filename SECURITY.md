## Reporting a Vulnerability

The security of our project is a top priority. We appreciate your efforts to responsibly disclose your findings and will make every effort to acknowledge your contributions.

### How to Report

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report security vulnerabilities using one of the following methods:

1. **GitHub Security Advisory (Preferred)**
   - Navigate to the [Security tab](https://github.com/yourorg/yourproject/security/advisories) of this repository
   - Click "Report a vulnerability"
   - Fill out the advisory form with details

2. **Email**
   - Send an email to: security@yourcompany.com
   - Use PGP key (fingerprint: `1234 5678 90AB CDEF 1234 5678 90AB CDEF 1234 5678`) for sensitive information
